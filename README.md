# Система внутренних голосований для сотрудников университета

Веб-платформа, предназначенная для работников университета. Позволяет:

- Создавать голосования по различным вопросам  
- Принимать участие в голосованиях с любого устройства  
- Просматривать результаты в удобном формате  

Проект состоит из двух частей:
- **`backend`** — API для управления голосованиями, пользователями и результатами  
- **`frontend`** — веб-интерфейс для создания, участия и анализа голосований
